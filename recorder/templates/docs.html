BotFlow is a framework for creating web automation bots that auto-heal broken selectors using LLM vision. Record flows visually, parameterize values, and deploy headless.

How it works

⚫
Step 1

Record

Navigate to any website. Click elements, fill forms, extract data. Every action is captured as a replayable step.

✎
Step 2

Edit & Parameterize

Convert hardcoded values into template parameters. Name your extracted variables. Adjust selectors and timeouts.

▶
Step 3

Run & Extract

Execute flows with parameters. Get structured data back. Run headless in production or visible for debugging.

Features

Built for reliability on sites that change constantly.

⚙

LLM Auto-Heal

When a selector breaks, the engine captures a screenshot and DOM snapshot, sends them to Claude, and gets a repaired selector. Fixes are persisted to the flow JSON.

{{}}

Parameterized Flows

One-click conversion of hardcoded values to {{params.name}} templates. Reuse the same flow with different inputs.

☷

Context-Anchored XPath

Selectors are anchored on nearby text labels, not fragile positional indices. The recorder walks up the DOM to find the meaningful section title.

☼

Headless & Visible

Run visible for recording and debugging, switch to headless for production. One toggle in the UI or a parameter in the API.

API

Use BotEngine as a Python library in three lines of code.

bot.py Python 3.11+ 
from botengine import BotEngine

async def main():
    # Initialize the engine
    engine = BotEngine(flows_dir="flows/", headless=True)
    await engine.start()

    # Run a flow with parameters
    result = await engine.execute(
        "Betclic extract game info",
        params={"url": "https://betclic.fr/match/..."}
    )

    # Access structured results
    print(result.returns)
    # {"home_team": "PSG", "away_team": "OM", "score_home": "2", ...}

    await engine.stop()
